2023 Applied Information Engineering Hackathon.

Created with a time limit of 24 hours with two peers. 
The objective was to create a program that helps manage GPA and credits needed to graduate.
The problem itself was simple; this was not an algorithm question. So my teammates and I kept our focus on UI/UX. 
While other teams were confined to terminal output, we utilized the streamlit library to create a local website. 
The main features were GPA calculator, a graduation requirement checker, and the ability to upload and download excel files 
that contain class/grade data. 

Won first place.
